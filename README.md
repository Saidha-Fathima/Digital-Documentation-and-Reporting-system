# Digital Documentation & Reporting System

A comprehensive system for digital documentation and job reporting, featuring a Flask backend and a modern React frontend.

##  Quick Start

For detailed setup instructions, please refer to the [Setup Guide](./SETUP_GUIDE.md).

### Backend
```bash
cd backend
python -m venv venv
# Activate venv then:
pip install -r requirements.txt
python app.py
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```

## Tech Stack
- **Backend:** Flask, SQLite, Flask-CORS, Flask-Session
- **Frontend:** React 19, Vite, Tailwind CSS 4, React Router
- **Icons:** Heroicons
